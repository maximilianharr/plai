# Docker Containers

Each main docker has a separate folder / planet with detailed documentation (README.md).

## Docker Cheatsheat

### Show all containers
```bash
docker ps -a
```

### Remove all containers
```bash
docker stop $(docker ps -a -q)
docker rm $(docker ps -a -q)
```
